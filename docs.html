Introduction

Welcome to the YourDevTools.io API documentation. Our platform provides a powerful set of tools to help you build, test, and deploy your applications with ease. Our RESTful API is designed to be intuitive and follows modern standards.

Authentication

All API requests must be authenticated with a valid API key. You can find your API key in your account dashboard. Include your key in the Authorization header for every request.

curl -H "Authorization: Bearer YOUR_API_KEY" https://api.yourdevtools.io/v1/projects

API Endpoints

Here is a list of our primary API endpoints. For more detailed information on parameters and responses, see the full API reference.

  • GET /v1/projects - Retrieve a list of all your projects.
  • POST /v1/projects - Create a new project.
  • GET /v1/projects/{id} - Get details for a specific project.

Data Models

Our API uses standard JSON data models. Here's an example of a typical project object.

{
  "id": "proj_12345",
  "name": "My Awesome Project",
  "status": "active",
  "created_at": 1672531200
}
